# Hotkeys

Under the hood the library uses [reakeys](https://github.com/reaviz/reakeys) for
binding hotkeys.

### Camera Controls

```ts
{
  name: 'Zoom In',
  keys: 'mod+shift+i'
},
{
  name: 'Zoom Out',
  keys: 'mod+shift+o'
},
{
  name: 'Center',
  keys: ['mod+shift+c']
}
```

### Selection

```ts
{
  name: 'Select All',
  keys: 'mod+a',
},
{
  name: 'Deselect Selections',
  keys: 'escape'
}
```
