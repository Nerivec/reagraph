# NextJS
For NextJS, you need to use the `use client` directive to make the component client side.
If you don't, the reagraph will render only a placeholder on the server side.

## Importing
To avoid issues with the context, you need to import the components, hooks and symbols separately.

```jsx
// ✅ Components only (no hooks, no context issues)
import { GraphCanvas, GraphScene } from 'reagraph';

// ✅ Hooks only (when you need them)
import { useCameraControls, useCenterGraph } from 'reagraph/hooks';

// ✅ Symbols only (when you need them)
import { Node, Edge, Arrow } from 'reagraph/symbols';
```

## Example
```jsx
'use client';
import { GraphCanvas } from 'reagraph';

export const SimpleGraph = () => {
  return (
    <div className='h-[300px] w-full relative'>
      <GraphCanvas
        labelType='all'
        nodes={[
          {
            id: 'n-1',
            label: '1',
          },
          {
            id: 'n-2',
            label: '2',
          },
        ]}
        edges={[
          {
            id: '1->2',
            source: 'n-1',
            target: 'n-2',
            label: 'Edge 1-2',
          },
        ]}
      />
    </div>
  );
};

```
